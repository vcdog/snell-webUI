# 暗黑模式 - 快速开始

## 🎨 功能简介

Snell Panel 现在支持暗黑模式！您可以：
- 🌞 使用**浅色模式**（适合白天）
- 🌙 使用**深色模式**（适合夜间）
- 🔄 使用**自动模式**（跟随系统）

## 🚀 如何使用

### 切换主题

1. 找到右上角的主题按钮
   - 主界面：在 GitHub/Telegram 图标旁边
   - 登录页：在右上角浮动位置

2. 点击按钮切换主题
   - 第1次点击：自动 → 浅色
   - 第2次点击：浅色 → 深色
   - 第3次点击：深色 → 自动

3. 查看当前模式
   - ☀️ 太阳图标 = 浅色模式
   - 🌙 月亮图标 = 深色模式
   - ⚪ 组合图标 = 自动模式

### 自动模式

设置为"自动模式"后：
- 白天自动显示浅色主题
- 晚上自动显示深色主题
- 跟随系统设置，无需手动调整

## 📋 查看效果

刷新页面（`Ctrl + Shift + R`）后即可看到：

| 模式 | 背景色 | 文字色 | 适用场景 |
|------|--------|--------|---------|
| 浅色 | 白色 | 深色 | 白天使用 |
| 深色 | 近黑 | 浅色 | 夜间使用 |
| 自动 | 跟随系统 | 自动调整 | 24小时 |

## ⚙️ 设置系统主题

### Windows 11/10
1. 打开"设置" → "个性化" → "颜色"
2. 选择"浅色"或"深色"模式

### macOS
1. 打开"系统偏好设置" → "通用"
2. 选择"外观"：浅色或深色

### Linux (GNOME)
```bash
# 深色模式
gsettings set org.gnome.desktop.interface gtk-theme 'Adwaita-dark'

# 浅色模式
gsettings set org.gnome.desktop.interface gtk-theme 'Adwaita'
```

## 💾 偏好保存

您的主题选择会自动保存，下次访问时自动应用。

## 🎯 推荐设置

- **日常使用**: 选择"自动模式"
- **白天工作**: 选择"浅色模式"
- **夜间使用**: 选择"深色模式"

## 常见问题

### Q: 主题不自动切换？
A: 确保选择了"自动模式"（⚪ 组合图标）

### Q: 如何永久使用深色模式？
A: 点击主题按钮切换到 🌙 月亮图标即可

### Q: 颜色显示异常？
A: 清除缓存并强制刷新（`Ctrl + Shift + R`）
